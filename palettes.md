# QuietType Palettes

These are palettes for **[QuietType](https://www.wmprkr.xyz/quiettype/)**.

## 🖋 Monochrome Signal

*A near-black ink world with just enough wavelength shift to guide the eye.* ***The default QuietType experience.***

```css
/* Monochrome Signal */

  --bg: #141414;
  --text: #e2e2e2;
  --accent: #8f9fb0;
```
